# How to Run the Mobile App

The mobile version of our TMDB project is located in the `/mobile` folder.

To run the app:

1. Open a terminal
2. Navigate into the mobile folder:
   cd mobile
3. Install dependencies:
   npm install
4. Start the Expo development server:
   npx expo start

This will open Expo Dev Tools.  
You can run the app using:

- Expo Go on your phone
- An Android emulator
- An iOS simulator (Mac only)
